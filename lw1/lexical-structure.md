## Конспект
### Описание лексической структуры С++ и Python.

1. Элементы структурного программирования:
* блоки кода
* ветвления
* циклы

###### Python:
* Блок начинается с двоеточия (:) после заголовка конструкции (if, for, def и т.д.).
* Содержимое блока выделяется отступом (обычно 4 пробела).
* Отступы — лексемы: интерпретатор генерирует виртуальные токены INDENT и DEDENT (как скобки в других языках).
``` python
if x > 0:
    print("positive")
    y = x * 2
```
* Единственная конструкция: if–elif–else.
* Условие записывается без скобок: if expr:
* elif — ключевое слово (сокращение от else if).
* switch отсутствует в классическом виде. Начиная с 3.10 есть match/case, но это не jump-table, а pattern-matching по структуре данных — лексически и семантически иная конструкция.
###### C++:
* Блок заключается в фигурные скобки { }.
* Отступы не влияют на синтаксис — только для читаемости.
* { и } — настоящие лексемы (токены).
```cpp
if (x > 0) {
    std::cout << "positive";
    int y = x * 2;
}
```
* if (expr) stmt else stmt — скобки вокруг условия обязательны.
* else if — не ключевое слово, а else + вложенный if.
* switch (expr) { case const_expr: ... } — работает только с целыми типами и enum.
* scase заканчивается двоеточием (:), а не фигурной скобкой.

```В Python лексер должен отслеживать стек уровней отступа и генерировать INDENT/DEDENT при изменении глубины. В C++ — нет: скобки явно размечают границы. Это делает лексер Python сложнее.```
